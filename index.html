Featured Project

Pick N' Choose

Pick N' Choose is an Android application built using Java. Using the Yelp REST api, Pick N' Choose will decide on a food, drink, or cuisine for the user to eat, as well as display nearby restaurants which serve the selected food. Pick N' Choose also allows for the user to create a custom list of items to choose from instead of the included lists.
